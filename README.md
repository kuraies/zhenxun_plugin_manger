# 真寻插件管理
# 插件管理指令使用说明

---

## 1️⃣ 列表
**命令**：  
插件管理 列表

**功能**：显示所有已加载插件状态  
**示例**： 插件管理 列表


---

## 2️⃣ 未加载插件
**命令**：  
插件管理 未加载插件

**功能**： 显示未加载的插件列表  
**示例**： 插件管理 未加载插件


---

## 3️⃣ 加载插件
**命令**：  
插件管理 加载 `[plugin名称]` `[-D 插件ID]` `[-P 模块路径]`

**功能**：加载指定插件  
**参数说明**：
- `plugin`：插件名称（可选，优先按名称加载，例如 `music`）  
- `-D` / `--id`：插件 ID（可选）  
- `-P` / `--path`：插件模块路径（可选，例如 `zhenxun.plugins.music`）  

**注意**：
- 如果同时提供多个参数，按命令中出现的顺序优先匹配  

**示例**：
```
插件管理 加载 music
插件管理 加载 -D 22
插件管理 加载 -P zhenxun.plugins.music
```


---

## 4️⃣ 卸载插件
**命令**：  
插件管理 卸载`[plugin名称]` `[-D 插件ID]` `[-P 模块路径]`

**功能**：卸载已加载插件  
**参数规则** 同“加载插件”  

**示例**：
```
插件管理 卸载 music
插件管理 卸载 -D 22
插件管理 卸载 -P zhenxun.plugins.music
```


---

## 5️⃣ 重载插件
**命令**：  
插件管理 重载 `[plugin名称]` `[-D 插件ID]` `[-P 模块路径]`

**功能**：先卸载再加载插件  
**参数规则** 同“加载插件”  

**示例**：
```
插件管理 重载 music
插件管理 重载 -D 22
插件管理 重载 -P zhenxun.plugins.music
```

---

### ⚡ 注意事项
1. 插件 ID 为 `get_noload_plugins()` 或数据库中返回的唯一数字 ID  
2. 模块路径必须是 Python 模块路径，如 `zhenxun.plugins.music` 或 `zhenxun.builtin_plugins.web_ui`  
3. 名称 / ID / 路径冲突时，按用户输入的顺序优先匹配  
4. 未加载插件可以通过 `插件管理 未加载插件` 查询
